Instalación de Packet Tracer

Paso 1: Descargar Packet Tracer

  • Para Windows 10/8.1/8/7: Descargar Cisco-PT-620.exe
  • Para Linux (Debian, Ubuntu, Linux Mint): Descargar Cisco-PT-620.tar.gz

Paso 2 (Linux): Preparar el equipo

Si tu S.O Linux es de 32 bits, salta este paso. Si tu S.O es de 64 bits debes ejecutar lo siguiente para habilitar la compatibilidad con arquitectura de 32 bits:
$ sudo dpkg --add-architecture i386
$ sudo apt-get install libnss3-1d:i386 libqt4-qt3support:i386 libssl1.0.0:i386 libqtwebkit4:i386 libqt4-scripttools:i386

Paso 3 Instalación:

  • En Windows se instala como programa común y corriente
  • En Linux debemos dirigirnos a la carpeta donde se guardó la descarga y ejecutar:
    • $ tar -xvf Cisco-PT-620.tar.gz
    • $ cd PacketTracer62Student
    • $ ./install

Paso 4: Ejecución

  • En Windows: buscar el programa y ejecutar con doble click
  • En Linux: lo mismo que en Windows o desde el terminal ejecutar packettracer
